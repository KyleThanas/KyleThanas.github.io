# 地板or瓷砖

# 地板的5大优势和4大劣势分析


地板的脚感仅次于地毯，普通家庭不适合用地毯，但已经非常舒服了，而且地面不会很滑，不易摔倒，且就算摔倒也不会出现摔伤。

优势

劣势

贴近自然的审美感觉，价值感强。受天气及湿度影响较大，容易变形。
比较好的舒适感，脚感极佳。会释放甲醛，环保性稍差。
防滑效果好，不易摔伤。虽然可以地暖使用，但问题比较多。
施工简便，安装成本低。使用寿命相对较短。
易拆装，方便更换。



# 瓷砖的6大优势和4大劣势分析


瓷砖的标价只是材料，并不包含施工，所以做预算的时候一定要注意，千万别误以为瓷砖店里的标价就是全部了。相比家装用的很多材料，瓷砖是比较环保的，铺贴瓷砖用的水泥砂浆也很环保，这点优势很大。


购买地砖时，有一个服务一定要注意：能不能送货上楼！没感受过瓷砖重量的可能体会不到，千万别小看！

优势

劣势

容易清理，保养简单。

过于坚硬冰冷，体感不好，且容易摔伤。
无有毒有害污染，不会造成空气污染。个别瓷砖会有放射性污染。
使用寿命长，一般可以使用10-20年。铺装复杂、施工繁琐。
防火、防水、防腐性能好。后期拆除难度大，成本高。
花色极其丰富，可选范围大。


配合地暖使用，且不用担心打坏水管。


# 瓷砖VS地板：成本、保养


# 比成本：地板略胜一筹

地砖因为铺贴成本比较高，所以价格就会贵一些。整体来说，以完工来算，中高档地砖的价格和实木地板的价格相仿，强化地板则要便宜很多。


从使用角度来说，强化地板和实木地板并没有什么本质性的差异，而且强化地板更便宜，更容易打理，所以整体销售情况来看，还是强化地板更好一些。


现在地砖虽然尺寸更大（从600变成了800），表面抛釉工艺更先进（各种迭代技术，更透，更亮，更致密），但材料价格实际上要比几年前更低。


# 比保养：瓷砖扳回一局

瓷砖唯一可能出问题的是两种情况：1）砖缝发黑；2）出现碎裂。


前者可以通过使用好的美缝剂解决，后者属于小概率事件。


地板要考虑的保养就复杂的多，怕水、怕干、怕磨，尤其是潮湿地方使用寿命会大受影响。


地暖专用地板有很多，但原理和效果存疑。地板材质原因，有一定隔热作用，不利于地暖热量散发。如果是强化地板很容易在地暖烘烤下加速甲醛挥发（也就是加速胶的分解），污染会加重，且会影响寿命。部分通过龙骨安装的地板，安装时还很容易打漏地暖管线，损失严重。


# 选哪个


# 地暖

首先，是不是地暖房子，如果是好吧，你可以铺地砖，也可以铺木地板，但是木地板只能用强化或者复合，纯实木是绝对用不成的。


加个题外话，现在好多木地板商家声称他们有新产品，就是不怕地暖的纯实木地板，反正我是不信，比如我给你说，我有不怕火的汽油，然后拿一盆水做实验。地板商声称的也就是这个猫腻，其实压根不是什么不怕地暖的纯实木，就是作假而已。


# 光线

你家光线好吗？如果很好，那就随便，可以贴砖可以铺木地板，如果光线不好，建议不要木地板，否则不反光，房子会更暗。


# 老人和小孩

如果有老人在住，或者有小孩在住，这里的小孩说的是四五岁以及以下。你只能铺瓷砖，绝对不能铺木地板。
