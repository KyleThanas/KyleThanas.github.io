# 红酒和葡萄酒有什么区别

# 红酒≠葡萄酒

**
葡萄酒就是用葡萄酿的酒，红酒是指红葡萄酒。
全世界近1万种葡萄，除了少数例外果肉和果汁基本都是白的。红酒酿制过程开始的时候，会在葡萄汁里保留红葡萄皮进行浸皮发酵，这样红葡萄皮里的单宁和红色素就会跑到葡萄汁里（也有一部分来自于葡萄梗和葡萄籽）。 
“红酒”只是“葡萄酒”的一种。


打个比方，我们习惯见到男人就喊“帅哥”，但男人不只有帅哥，还有普通哥、二逼哥……

同样，葡萄酒不只有“红酒”，还有白葡萄酒、桃红葡萄酒


# “赤霞珠”是什么酒？


赤霞珠不是酒，是葡萄品种的名字。
“品种”可以告诉我们很多信息，比如：

同样，在打开一瓶酒之前，我们也能从葡萄品种得到很多信息：

在哪儿看葡萄品种呢？
新世界酒庄，或者旧世界的新派酒庄，会在酒标的明显位置标明：

而旧世界的许多传统酒庄，会把产区放在十分明显的位置：

要了解这类酒的葡萄品种，就需要更渊博的产区知识，比如“夏布利”产区的白葡萄酒，基本都是用“莎当妮”这种葡萄酿成的。

# 红酒养生吗？

红酒说到底还是酒，最重要的成分是酒精

红酒喝完的主要作用是开心，要养生，还不如吃葡萄不吐葡萄皮……

# “干红”是个啥意思？

**“干”就是不甜的意思，"干红"是每升残糖量不超过4克的红酒。

# 挂杯的都是好酒吗？

当然不是。打个比方，胖子都是有钱人吗？
在以前大家都吃不饱的时候，胖代表着有钱。

现在大家都能吃饱，还有人觉得胖代表有钱吗？

以前酿酒技术不成熟，挂杯代表好酒。
现在所有的红酒挂杯都不错，可别再以挂杯论好坏了。

# 瓶底深的都是好酒吗？

好酒能存几十年，会产生很多沉淀，所以需要一个深瓶底。

但当大家都认为瓶底深就是好酒时，有些酒庄出于营销目的，什么酒都用特别深的瓶底……
所以，大侠带刀，但带刀的不一定是大侠


懂了吧？

# 橡木塞的红酒一定比螺旋盖的好吗？

不一定啊。
以前材料技术不发达，酒塞的材料只能从自然界找，橡木塞是最能让酒微氧“呼吸”的：

随着材料科技的发现，螺旋塞加入了高分子材料，同样能让酒微氧“呼吸”，而且能控制透氧率：

另外，用于制造橡木塞的栓皮栎树，主要种植在西地中海和中国秦岭地区，南半球的新世界产酒国拼了命的研究螺旋塞技术，也是因为离得太远不包邮啊～

# 法国红酒最好吗？

如果把法国看成一家大公司，这家公司产品好，又靠近大客户（英国皇室），所以在相当长的时间里称霸世界

后来大客户穷了，不是“日不落”了，加上大公司发现机会越来越少（可开发的土地越来越少），于是高管们纷纷跳槽到有前途（产区气候条件同样好）的创业公司，创业公司们就蓬勃发展起来了

# 背标上有中文字就是国内灌装的?

日本动漫要是没有中文字幕，大部分人也看不懂
出于同样的目的，为了让消费者看懂酒的信息，我国法律规定，进口红酒过海关时必须加贴中文背标：

# 自家酿的红酒应该比较放心吧？

自家酿的红酒，用的啥原料自己都知道，所以感觉好像会觉得放心一些：

可是，自家酿酒时无法控制微生物环境，酿出的酒可能有甲醇和各种杂醇

这些物质喝下去轻则头疼，重则影响视力，还是少喝为好

# 这瓶酒的条码开头是69，是国内罐装的

条码的前缀码只代表条码注册地，不代表原产国。
就好像有身份证只代表有中国国籍，不代表一定在中国出生。
怎么确定原产地比较靠谱呢？看中文标签上的“原产地”一项
